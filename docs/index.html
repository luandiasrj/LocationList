Como Usar o Código da Cidade

1. Encontre o Código

Use a ferramenta de busca acima para encontrar o código da sua cidade.

2. Copie o Código

Clique no botão de cópia ao lado do código da cidade desejada.

3. Configure seu Relógio

Abra as configurações do seu relógio e cole o código na configuração de previsão do tempo.

Exemplo de Configuração

A maioria dos relógios chineses (como GeekMagic SmallTV-Pro) com previsão do tempo usa a API QWeather. Para configurar a previsão do tempo, você precisará inserir o código da cidade no aplicativo do relógio.

Passos típicos:

  1. Abra o hotspot do seu relógio, irá aparecer o IP na tela.
  2. Vá para as configurações do relógio
  3. Encontre a opção de previsão do tempo ou clima
  4. Insira o código da cidade que você encontrou neste site
  5. Salve as configurações e sincronize seu relógio

Perguntas Frequentes

QWeather é um serviço de previsão do tempo usado por muitos relógios inteligentes chineses, incluindo dispositivos GeekMagic. Cada cidade tem um código único que o relógio usa para buscar a previsão do tempo para aquela localidade.

Muitos relógios inteligentes chineses não conseguem detectar automaticamente sua localização ou não têm uma interface para buscar cidades pelo nome. Em vez disso, eles exigem que você insira manualmente o código da cidade para configurar a previsão do tempo.

Se sua cidade não estiver na lista, tente buscar por uma cidade maior próxima. Cidades pequenas podem não estar disponíveis na API QWeather. Alternativamente, você pode tentar buscar pelo nome da região ou município.

Verifique se você inseriu o código correto. Alguns relógios podem exigir um formato específico ou ter limitações. Certifique-se também de que seu relógio está conectado ao Wi-Fi e que o aplicativo tem permissão para acessar a internet. Se o problema persistir, tente reiniciar o relógio e as configurações segurando o botão de reset por alguns segundos.
